# Filtre de Kalman pour l’estimation de la position d’un mobile en 1D

## Description
Ce projet présente l’implémentation d’un filtre de Kalman permettant d’estimer la position d’un mobile se déplaçant sur un axe unidimensionnel (1D).

L’objectif est d’améliorer la précision des mesures en réduisant l’influence du bruit présent dans les données des capteurs.

## Structure du projet
- `rapport filtre de Kalmandocx` : contient le rapport détaillé du projet.
- `rapport filtre de Kalman` : contient le code source MATLAB du filtre de Kalman.

## Exécution du code
1. Ouvrir MATLAB.
2. Charger le fichier `rapport filtre de Kalman.m`.
3. Exécuter le script pour lancer la simulation et visualiser les résultats du filtre de Kalman.
